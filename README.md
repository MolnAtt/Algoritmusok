# Algoritmusok jegyzet

## Listák programozási tételei

Az elemi listakezelési feladatok két fő csoportra bonthatók:

- Amikor végig kell nézni a listát: 
    - Összegzés (sum)
    - Maximumkeresés (max)
    - Másolás (map, SELECT)
    - Kiválogatás (filter, WHERE)
- És amikor nem: 
    - Eldöntés
    - Keresés
